# Scope Rules

## Prototype must-have

- 6x6 grid
- place/remove/move items
- ingredients
- machines
- recipes
- contracts
- economy
- upgrade choices
- 5-contract run
- run summary
- tests

## Prototype nice-to-have

- seeded runs
- simple hover tooltips
- simple visual highlights
- basic local save
- keyboard shortcuts
- recipe book panel

## Post-validation

- final art
- sound
- Steam page
- Steam demo
- trailer
- expanded recipes
- meta progression
- daily challenge
- mobile port
- DLC concepts

## Scope creep

- 3D
- character creator
- story campaign
- world map
- relationship system
- online leaderboard
- multiplayer
- crafting hundreds of items
- complex procedural generation
- live-service events

## Dangerous/disallowed for v0.1

- live AI gameplay
- backend
- accounts
- payments
- mobile ad SDKs
- IAP
- blockchain/NFTs
- gambling mechanics
