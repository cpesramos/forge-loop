# Product Brief — Forge Loop

## Working title

Forge Loop

## Genre

Grid-based crafting roguelite / systems puzzle roguelite.

## One-sentence pitch

Place ingredients and forge machines on a compact grid to fulfil magical crafting contracts across short, replayable roguelite runs.

## Target player

Players who like:

- Balatro-style replayable systems
- Backpack Battles-style spatial decisions
- Stacklands-style card/item logic
- Luck Be a Landlord-style compact runs
- optimisation puzzles
- crafting chains
- roguelite upgrades

## Target platform

Prototype:
- Web

Commercial target:
- Steam

Possible later:
- mobile port
- Switch/console port

## Monetisation

Prototype:
- none

Commercial:
- Steam premium
- possible later DLC if successful

Avoid:
- mobile ads
- mobile IAP
- subscriptions
- live-service monetisation

## Core promise

Every run should create small optimisation decisions:

- Which contract do I pursue?
- Which machines do I buy?
- How do I arrange the grid?
- Which upgrade changes my strategy?
- Can I squeeze one more crafted output from limited space/resources?

## MVP content

- 6x6 grid
- 6 ingredients
- 4 machines
- 10 recipes
- 5 contract types
- 8 upgrades
- 5-contract run
- ugly but usable UI

## Non-goals

- final art
- audio
- animation polish
- multiplayer
- backend
- live AI
- daily mode
- Steam page
- mobile UI
- monetisation
